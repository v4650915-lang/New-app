# История изменений (Changelog)

Все заметные изменения в этом проекте будут описаны в этом файле.

## [Невыпущенное]

### Добавлено
- Новый модуль "Калькулятор Сферы" в панель инженера (EngineeringPanel).
- Файл `utils/math.ts` с математической логикой для расчета профиля сферы (прогиб, 4 точки сопряжения).
- Компонент `SphereApp.tsx` с полноэкранным UI, адаптированным под общий Fanuc-стиль (темные тона, SVG-визуализация, таблица точек).

### Изменено
- Обновлен `App.tsx` для поддержки полноэкранного режима нового модуля "Сфера" по аналогии с модулем "Симулятор".
- Обновлен `EngineeringPanel.tsx` — добавлена 5-я кнопка "СФЕРА".

### Исправлено
- Логика калькулятора сферы: восстановлен корректный вывод 4-х точек сопряжения (вместо обрезанных 2-х) с отрицательными значениями по оси Z.
